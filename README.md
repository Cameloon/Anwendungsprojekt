### Bun Commands

    To install dependencies:
    ```bash
    bun install
    ```

    Development - full-stack dev server with hot reload
        bun dev


    Static Site - build optimized assets to disk (no backend)
        bun run build


    Production - serve a full-stack production build
        bun start
